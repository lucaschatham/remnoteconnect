## Review Policy

- Safety-sensitive pull requests require maintainer review.
- Changes to auth, token handling, host/origin checks, graph-wide mutation, hard delete, backup/undo behavior, or plugin permissions require tests.
- Public examples and issue reports must not include private RemNote content, daemon tokens, backups, exports, or generated graph maps.
- The project is MIT open source, but npm publishing and RemNote marketplace packaging are not active in v0.3.
