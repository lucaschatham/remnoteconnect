# Code of Conduct

RemNoteConnect uses the Contributor Covenant Code of Conduct, version 2.1.

Read the full text at:

https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Reporting

Report unacceptable behavior to the project maintainer through GitHub. If the report includes security-sensitive details, daemon tokens, private RemNote content, or private graph data, use a private GitHub security advisory instead of a public issue.
